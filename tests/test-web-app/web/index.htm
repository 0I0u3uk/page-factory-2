Test Automation Gears

TPage-Factory - фреймворк для автоматизированного тестирования через GUI(graphical user interface, графический пользовательский интерфейс), можно использовать для Web или Android приложений.

Api-Factory - фреймворк для автоматизированного тестирования через API(application programming interface, программный интерфейс приложения)

Mq-Factory - фреймворк для автоматизированного тестирования MQ(messages queue), поддерживается работы со стандартом JMS и Apache Kafka
