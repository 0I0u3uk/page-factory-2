== HTML Plugin
Инструмент, позволяющий реализовать web автоматизацию на проекте с использованием типизированных элементов и блоков. Он базируется на `selenium`,  `cucumber` и `htmlelements-java`. Инструмент содержит логику взаимодействия с блоками и типизированными элементами

:plug: html-plugin
:pn: HTMLPage
:st: HtmlSteps
:stg: HtmlGenericSteps
:act: HtmlPageActions
